# bonzo v2

pieza web interactiva para fragmentos aleatorios de bonzo.

## archivos

- `index.html`: estructura principal
- `style.css`: diseño visual y responsivo
- `script.js`: lógica, animación y sonido generativo
- `versos.json`: versos editables

## publicar en github pages

sube estos 4 archivos a tu repositorio y activa:

settings → pages → deploy from a branch → main → /(root)
